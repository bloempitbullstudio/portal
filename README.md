# Bloempitbull Studio – V2 met Owner/Dev Panel

**Nieuw in deze versie**
- Registreren vraagt nu ook **e‑mail** (wordt opgeslagen bij de user).
- **Owner/Dev panel** (onderaan `aanvraag.html`) – alleen zichtbaar voor `Bloempitbull`:
  - Rollen wijzigen (`user` / `dev` / `owner`)*
  - E‑mail aanpassen
  - **Wachtwoord resetten** (hash wordt opgeslagen; bestaande wachtwoorden worden nooit getoond)
- Footer‑jaartallen automatisch `2025‑<huidig_jaar>`.

*Primary owner (`Bloempitbull`) kan **niet** naar een andere rol gezet worden om lock‑out te voorkomen.
