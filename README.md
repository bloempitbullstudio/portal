# Bloempitbull Studio – Roblox Game Portal

Bestanden:
- `index.html` – inloggen/registreren (owner automatisch)
- `aanvraag.html` – verzoeken indienen/beheren (owner kan statussen wijzigen)
- `favicon.png`, `apple-touch-icon.png` – iconen (vervang ze gerust door je eigen logo met **dezelfde bestandsnamen**)

## Footer jaartal
De footer toont automatisch `2025-<huidig_jaar>`.

## Publiceren op GitHub Pages (kort)
1. Nieuwe GitHub‑repo → upload alle bestanden in de root.
2. **Settings → Pages** → **Deploy from a branch** (branch: `main`, folder: `/`).
3. Wacht ±1 minuut → je site staat live op `https://<jouw-naam>.github.io/<repo>`.
